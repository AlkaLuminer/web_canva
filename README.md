### How to use 

    First, introduce the buttons in the top label.
    ↩ Undo: undo button, including modification of pen, eraser, shapes and image.
    ↪ Redo: redo button, including modification of pen, eraser, shapes and image.
    🌅 Add Image: upload image (jpg, png, webp file)
    −: zoom out.
    +: zoom in.
    
    Additionaly, by using mouse's scrolling, it can zoom in/out.

    🏠 Home: set the zoom and canva position to original place.
    ⬇ Save: Download canva
    🎵: turn on/off play background music(currently set L tree Do)
    🔊: turn on/off sounds of drawing.
    ⚙ Size: set the canva's height and width.
    Layer 1/2/3: switch layers.
    ↺: reset it's layer(including drawings and image on this layer).

    Then, on the left side tools bar:

    tools:
    ✏️: pencil mode for drawing, settings are color, brush shape, size, opacity parts.
    🧽: eraser mode for drawing, setting size with the same one as pencil.
    T: text input mode for drawing, click in the canva then type, use mouse to select different typefaces, and then press Enter to set the text, setting size and color with the same one as pencil. 
    ▭: rectangle mode for drawing, click on canva first then drag its height and width to modify, and then click it again to set, setting size and color with the same as pencil, also can fill inside by clicking Fill Shape first then draw rectangle.
    ○: cirle mode for drawing, functioning and settings are the same as rectangle.
    ╱: straight line mode for drawing, functioning as revtangle, settings are the same as pencil.
    △: triangle mode for drawing, functioning and settings are the same as rectangle.
    ⊹: selecting mode for moving images to other places, different images on different layers should be moved or modified on its layer.
    ✋: draging canva mode for moving canva to a better position.

    color:
    Stroke: set drawing color, remember to click OK to set color after choosing one.
    Fill: set drawing color for shapes drawings, remember to click OK to set color after choosing one.

    Brush Shape: (for pencil mode)
    ●: circle brush.
    ■: square brush, better to discriminate with circle in larger size of stroke.
    ╱: calligraphy/line brush, better to discriminate with square in larger size of stroke. 
    ⁘: spary brush.

    Size: use the slide to control current pencil's size, shape's side's width, and eraser's size

    Opacity: use the slide to control current pencil's and shape's transparenty/Opacity.

    Fill Shape: click on/off to fill/unfill the current drawing shape.

    Quick Colors: functions the same as color Stroke but quicker to switch.
